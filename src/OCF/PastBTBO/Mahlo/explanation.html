① 什么是 Mahlo 宇宙? — 大基数阶梯

在数学里, 我们对"大"的概念有一个无限延伸的阶梯. 从有限数 → 可数无穷 ω → 不可数 ω₁ → ... 一直爬到所谓的 大基数 (large cardinals). 每往上一级, 数学强度都翻几个台阶.

点击下面阶梯上的任一级查看说明:

Mahlo 的本质: "反射"

一个基数 κ 是 Mahlo, 当且仅当 κ 下面的每一个 "稠密无界" (closed unbounded) 的不可达基数集合, 仍然包含一个不可达基数. 通俗一点: κ 这么大, 以至于你无论怎么从下面"取一族不可达基数", 总能在里面找到另一个不可达. 这种自我反射性把它推到比 Ω, Ω_Ω, ..., Ω_(Ω_Ω) 等都高的层级.

在类型论里, 对应物是 Mahlo 宇宙 (Mahlo universe): 对任意算子 f, 存在一个 sub-universe 在 f 下闭合. Setzer 1998 给出了第一个 IR-Mahlo 编码.

本项目的最终野心: 把 Mahlo 序数喂进 OCF 的 ψ 函数, 得到比 ψ(Ω_Ω) 更大的 collapsed countable ordinal — 即所谓的 ψ(M).

② Brouwer 化: 序数树长什么样?

经典集合论的序数是抽象的对象 (实际上是良序集的等价类). 但在构造性数学里, 我们要"亲手把序数搭出来". 最经典的搭法是 Brouwer 序数: 一棵树, 节点有三种.

data Ord : Set where
  zero : Ord                              -- 0
  suc  : Ord → Ord                       -- α+1
  lim  : (ℕ → Ord) → Ord                  -- sup of fundamental sequence

这里 lim f 就是"基本列 f 的上确界". 比如 ω = lim (n ↦ ⌜n⌝) 就是 0, 1, 2, ... 的极限.

Mahlo 加上去后是怎样的?

Mahlo 序数 Ordᴹ 在三种节点之外, 加上第四种 mahlo 节点:

data Ordᴹ : Set where
  zero  : Ordᴹ
  suc   : Ordᴹ → Ordᴹ
  lim   : (f : ℕ → Ordᴹ) → monoᴺ f → Ordᴹ
  mahlo : Opᴹ → (a : Ordᴹ) → (b : Sub a → Ordᴹ)
        → monoSub a b → (∀ s → a <ᴹ b s) → Ordᴹ

看起来吓人, 但是每个字段都对应一个直觉:

a : Ordᴹ

"code" 序数, 它对应的 sub-universe 是 Sub a.

b : Sub a → Ordᴹ

每个 sub-universe 元素 s ∈ Sub a 对应一个 reflected sub-universe b s.

monoSub a b

b 在 Sub a 的内禀序 _<ˢ_ 上单调. Phase 3 新增.

∀ s → a <ᴹ b s

strat 字段: code a 严格在每个 reflected sub-universe 下面. Phase 3 新增, 弱化代价.

Sub 是什么? — 每种节点的"宇宙内部"

Brouwer 树的每个节点都对应一个"我下面有哪些东西"的集合 Sub:

Sub zero = ⊥

0 下面没东西.

Sub (suc a) = ⊤

α+1 下面只有 α 一个新元素 (代理).

Sub (lim f _) = ℕ

极限节点下面是一族 f(n), 由 ℕ 索引.

Sub (mahlo _ a b _ _) = Σ (Sub a) (Sub ∘ b)

Mahlo 节点下面 = 一个 a 内的元素 s, 加上 b s 内的元素 t. 这是 Setzer Σ-反射的 Brouwer 化.

③ 8 层互递归的原理与作用

Mahlo 编码的核心技术活: 所有这些类型必须同时定义. Ordᴹ 提到 Sub, Sub 提到 Ordᴹ, mahlo 字段提到 monoSub, monoSub 又提到 _<ᴹ_... 它们循环依赖.

Agda 处理这个的方式叫 mutual recursion (互递归): 先声明所有类型, 再逐一定义. 当前 Phase 3 用 8 层互递归. 点击下面任一层查看它的作用:

点击上方任一层, 查看它的作用与依赖关系.

为什么需要这么多层?

关键洞察: 每一层都解决一个"循环依赖". 我们在尝试 V1 朴素编码 mahlo : (Ordᴹ → Ordᴹ) → Ordᴹ 时直接被 Agda strict positivity 拒. 然后:

  • Opᴹ 出现: 把 "Ordᴹ → Ordᴹ" 改走 "Ordᴹ ↔ Opᴹ 双向" (Takahashi MLQ 技巧).
  • Sub 出现: 给每个节点一个"我下面有什么"的可解码集合, 是 IR (induction-recursion) 的经典模式.
  • _<ᴹ_ 出现: 序数必须有 < 关系才能跑 ψ.
  • monoᴺ 出现: lim 不允许任意基本列, 必须单调, 否则下游 bounded trichotomy 不可证.
  • _<ˢ_ 出现 (Phase 3 新): Sub a 必须自带内禀序, 否则 mahlo 节点的 trichotomy 卡死. 详 §④.
  • monoSub 出现 (Phase 3 新): b : Sub a → Ordᴹ 不允许是任意函数, 必须在 _<ˢ_ 上单调.

④ 内禀序 _<ˢ_ 的奥秘

为什么必须有它?

设想 Phase 2 我们做 bounded trichotomy, 想证: 给定两个证明 p : a <ᴹ c 和 q : b <ᴹ c, 必有 a < b 或 b < a 或 a ≡ b.

当 c = mahlo f a₀ b₀ ... 时, p 和 q 可能是 mahlo-b s p' 和 mahlo-b s' q' 这种形式 — 即 "a 在 sub-universe b₀ s 内, b 在 sub-universe b₀ s' 内".

要继续证, 必须比较 s 和 s'. 但 Sub a₀ 没有序!

这就是 Phase 2 死墙的根源 — Sub a 是个被"解码"出来的集合, 它的元素之间没有任何天然顺序. mahlo 节点下面的两个名字 s, s' 在结构上根本不可区分.

解决方案: 按 a 的形状归纳定义 _<ˢ_

Phase 3 的关键发明: 给每种 a 节点分配一个"和它的 Sub 匹配的序". 用 IR-recursive 的方式定义:

_<ˢ_ : ∀ {a : Ordᴹ} → Sub a → Sub a → Set
_<ˢ_ {a = zero}        ()  _                -- Sub zero = ⊥, 空
_<ˢ_ {a = suc _}       _   _   = ⊥      -- 单点, 没有非平凡序
_<ˢ_ {a = lim _ _}     n   m   = n <ᴺ m  -- 借用 ℕ 的序
_<ˢ_ {a = mahlo _ a b _ _} (s , t) (s' , t') =
   (s <ˢ s') ⊎ Σ[ eq ∈ s ≡ s' ] (subst (Sub ∘ b) eq t <ˢ t')
                       -- 字典序 (lex order)

字典序: 互动演示

mahlo 节点的 Sub 元素是一对 (s, t) (第一分量在 Sub a 里, 第二分量在 Sub (b s) 里). 用字典序比较: 先比第一分量, 相等再比第二分量.

选择两对 (s, t) 和 (s', t') 比较:

s = 0 t = 1 vs s' = 0 t' = 2
比较 (0, 1) vs (0, 2): first 相同 (都是 0), 比 second: 1 < 2 → (0,1) <ˢ (0,2)

关键技术: subst 转移

注意 mahlo case 第二分量 t : Sub (b s), t' : Sub (b s') — 它们类型不同 (索引不同)! 当 s ≡ s' 时, 用 subst (Sub ∘ b) eq t 把 t 沿等式同构搬到 Sub (b s'), 然后才能和 t' 比较.

为什么这能通过 Agda 类型检查? 这里用 IR (induction-recursion) 的 Set-valued 函数定义 _<ˢ_. _<ˢ_ {b s'} 看起来不在 a 上"结构性变小", 但与 Phase 2 已有的 Sub (mahlo _ a b _ _) = Σ (Sub a) (Sub ∘ b) 同构 — 后者也有 Sub (b x) 不变小但被 IR scheme 接受. Agda 接受这种 IR 函数定义.

internal trichotomy on Sub

_<ˢ_ 自带无条件三歧性 <ˢ-dec: 任给 s, s' : Sub a, 总有 s <ˢ s' 或 s' <ˢ s 或 s ≡ s'. 证法: 结构归纳 a, mahlo case 套两层字典序展开. 这是 Phase 3 解锁 mahlo bounded trichotomy 的关键引擎.

⑤ Phase 1/2/3 的演进对比

Phase 1: 最小可行骨架

4 层互递归 (Ordᴹ + Opᴹ + Sub + monoᴺ). Sub 在 mahlo 节点上是占位的 ⊤ (Phase 2 替换). 目标: 验证 Takahashi MLQ 风格能通过 Agda strict positivity (V1 朴素编码、V3 Setzer 直译都被拒).

结果: ✓ 通过. 锁定 MLQ 是唯一可行编码路径.

Phase 2: 把骨架填满, 撞墙

升到 6 层: 加 _<ᴹ_, _<ᴼ_, 把 Sub (mahlo) 改成正确的 Σ-closure. 然后做 bounded trichotomy <ᴹ-dec.

非 mahlo case

  • ✓ zero/suc 平凡
  • ✓ lim/lim 用 ℕ 三歧 + monoᴺ trans

mahlo case (4 子 case)

  • ✓ (mahlo-a, mahlo-a)
  • ✗ (mahlo-a, mahlo-b)
  • ✗ (mahlo-b, mahlo-a)
  • ✗ (mahlo-b s, mahlo-b s')

退路

  • ⚠ 返回 Maybe
  • ⚠ 3 case 给 nothing
  • ✗ ψ_M 不可定

诊断: 3 个卡 case 都根源于 (a) Sub 无内禀序 (b) b 无 mono. 升级方向: Phase 3.

Phase 3: 加内禀序 + monoSub + strat

升到 8 层. 加 _<ˢ_ (IR 函数版), monoSub 字段, strat 字段 ∀ s → a <ᴹ b s.

mahlo case (4 子 case)

  • ✓ (mahlo-a, mahlo-a)
  • ✓ (mahlo-a, mahlo-b) — 用 strat
  • ✓ (mahlo-b, mahlo-a) — 用 strat
  • ✓ (mahlo-b s, mahlo-b s') — 用 monoSub + <ˢ-dec

解锁机制

  • ✓ <ˢ-dec 无条件三歧
  • ✓ monoSub 提供 b 单调
  • ✓ strat 桥接 a₀ ↔ b s
  • ✓ full <ᴹ-dec, 无 Maybe

代价

  • ⚠ strat 强制 a < b s 链式
  • ⚠ 不是 Setzer 真 Mahlo
  • ⚠ 弱化为 stratified Σ-reflect
  • ⚠ 强度 < 真 ψ(M)

⑥ 妥协与未来

本质交换 (Tradeoff)

Phase 3 用语义弱化 (strat 字段)换取操作性全函数 trichotomy. 这是构造性形式化中"忠实地翻译数学"和"实际能跑"之间的永恒冲突. 真正的 Setzer Mahlo 不要求 a < b s, 但那样 trichotomy 就证不出来.

三条候选路线

A. 去 strat 保 mono

退回 3a-only. cross-case 仍卡, partial trichotomy. 强度高但操作弱, ψ_M 不可定.

B. 替代 strat

找别的桥接 a ↔ b s 的方式 — 比如要求 Mahlo 节点带 reflection 见证. 研究问题, 未知答案.

C. 接受 stratified Mahlo (当前)

把当前编码标识为"弱 Mahlo OCF", 继续走 ψ_M 获取 ψ(stratified-Mahlo). 工程上可走, 强度可量化.

D. 换框架

用 Coq/Lean (它们的 strict positivity 检查不同) 重做. 离开 Agda 生态, 重写代价大.

⚠ 关键判断: Phase 3 ≲ Higher.agda

两者都是 "BTBO 之上 +1 级", 但增量方向不同:

  • Higher.agda 的增量在 "index 的内部结构": limₙ 让 Brouwer 树以"另一棵带 limᵢ 嵌套的 Brouwer 树"为索引. 索引集结构丰富.
  • Phase 3 mahlo 的增量在 "节点的外部结构": mahlo 节点带 Sub a 名字集. 但 Sub a 是 Σ-平铺, 没有内部 limᵢ 嵌套, 比 Ord ℓ 结构更扁.

Cardinality 等价, 结构非等价. Phase 3 嵌套 mahlo 大概率不超过 Higher.agda. 即 Phase 3 用 8 层互递归 + strat 弱化换来的强度, Higher.agda 已用 50 行更简单的方式达到了.

下一步: Step 5 ψ_M (定位为结构性贡献, 非强度贡献)

有了 full <ᴹ-dec, 现在可以仿 Higher.agda 里的 ψ<Ω 写出 ψ_M collapser. 但需诚实标识: 它的输出大概率不超过 Higher.agda 已有的 ψ((Ω_Ω)+(Ω_(ψ(Ω_Ω)+1))).

Phase 3 的真正价值不在 ordinal 强度提升, 而在:

  1. 探索 IR-Mahlo 的形式化路径 (结构性贡献).
  2. 为未来 "true Mahlo without strat" 的研究铺垫 (Phase 4+).
  3. 验证 monoSub + _<ˢ_ 的 IR-recursive 定义可行 (Dybjer-Setzer scheme 实证).
